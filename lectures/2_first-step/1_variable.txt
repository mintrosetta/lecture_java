
    :: Variable
    เป็นวิธีการเก็บข้อมูลไว้ในคอมพิวเตอร์ของเรา โดยข้อมูลสามารถเข้าถึงได้โดยใช้ ชื่อ ที่เราตั้งให้กับข้อมูล
    โดยข้อมูลเหล่านั้นจะถูกเก็บไว้บน random access memory (ram) โดยการสุ่มจากคอมพิวเตอร์

    :: Variable data type
        > int
            - จัดเก็บเลขจำนวนเต็ม

    :: Keyword
    เป็นคำใดคำหนึ่งในคำสงวนที่มีความหมาย และถูกกำหนดไว้ล่วงหน้าแล้วในภาษา Java

    :: Syntax
    จะเป็นแบบ case sensitive ตัวพิมพ์เล็กและใหญ่จะแตกต่างกัน เช่น int กับ Int จะแตกต่างกัน

    > Declaration Statement: int x = 5;
    > Initialize Statement: int x;
    > Assign Statement: x = 5;
